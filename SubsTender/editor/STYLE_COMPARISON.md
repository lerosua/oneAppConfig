# 卡片样式对比

## 🎨 前后对比

### 旧版样式（纯白色）
```
┌─────────────────────────────┐
│ 📦 服务名称          [删除] │  ← 白色背景 (95% 不透明)
├─────────────────────────────┤
│ [分类] [价格]               │
│                             │
│ 📊 3 个套餐                 │
│ 🔑 关键词: xxx              │
│ 🔗 访问网站                 │
│                             │
│ [编辑]                      │
└─────────────────────────────┘
```

**特点：**
- ❌ 纯白色背景，不透明
- ❌ 看不到背景渐变
- ❌ 缺少层次感
- ❌ 与背景对比太强

---

### 新版样式（毛玻璃效果）
```
┌─────────────────────────────┐
│ 📦 服务名称          [删除] │  ← 半透明 (15% 白色)
├─────────────────────────────┤  ← 背景模糊 20px
│ [分类] [价格]               │  ← 能看到背景渐变
│                             │  ← 白色文字
│ 📊 3 个套餐                 │  ← 轻微阴影
│ 🔑 关键词: xxx              │  ← 白色边框
│ 🔗 访问网站                 │
│                             │
│ [编辑]                      │
└─────────────────────────────┘
```

**特点：**
- ✅ 半透明背景，能看到渐变
- ✅ 毛玻璃模糊效果
- ✅ 白色文字清晰可见
- ✅ 与背景完美融合
- ✅ 现代化设计感

```css
边框: rgba(255, 255, 255, 0.18)  /* 半透明白边框 */
```

## 💎 Glassmorphism 设计原则

新版卡片遵循 Glassmorphism（毛玻璃拟态）设计原则：

1. **透明度** (Transparency)
   - 背景半透明，能看到后面的内容
   - 创造层次感和深度

2. **背景模糊** (Backdrop Blur)
   - 使用 backdrop-filter: blur()
   - 模糊背景，增加毛玻璃质感

3. **轻微边框** (Subtle Border)
   - 半透明白色边框
   - 定义卡片边界

4. **柔和阴影** (Soft Shadow)
   - 轻微的阴影效果
   - 提升立体感

5. **鲜明对比** (Vivid Colors)
   - 白色文字在渐变背景上清晰可见
   - 高对比度确保可读性
